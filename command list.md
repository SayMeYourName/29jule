
# Список основных команд для GIT
## Перечень команд
* 1. commit - сохранение результата работы с файлами;
- -m "позволяет добавить текстовое сообщение к команде";
- -a "сохраняет изменения во всех файлах папки".
* 2. git add - добавление файла;
- add . для отслеживания всех файлов в папке.
* 3. git status - просмотр состояния файла;
* 4. git log - просмотр журнала событий;
* 5. git checkout - проверка версии;
* 6. git diff - сравнение текущей и последней версиями.


*Справочно:*
*после завершения действий с файлом, все изменения сохраняются сочетанием клавиш "Shift+S" и последующими коммандами add и commit*
