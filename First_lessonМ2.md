## Выделение текста:
*Курсив.* - для выделения текста курсивом, его закрывают между * * (без пробелов между текстом и звездочками)

**Полужирный текст.** - для выделения текста полужирным шрифтом, его закрывают между ** ** (без пробелов между текстом и звездочками)

## Настройка и работа с удаленными репозиториями на GitHub


# Привязка удаленного репозитория на GitHub к Visual Studoi Code

- Привязанный удаленный репозиторий объявляем коротким именем "origin" для удобства работы с ним *"git remote add origin https://github.com/SayMeYourName/29jule.git"*

- Команда *"git branch -M main"* переименовывает ветку "master" в "main"

- Для добавления в удаленный репозиторий изменений (сведений) используем команду: *"git push -u origin main"*

- Командой *"git pull"* мы скачиваем  в локальный репозиторий изменения которые вносили в удалленом репозитории на GitHub

- Команда *"git clone "ссылка на удаленный репозитоий""* скачивает в локальный репозиторий данные с удаленного репозитория от другого пользователя через "Fork"

- Команда "cd" позволяет изменить дирректорию (папку) для работы в ней при получении данных (проекта) из удаленного репозитория

- При получении данных (проекта) другого пользователя для внесения изменений обязательно создаем ветку командой "git branch"
