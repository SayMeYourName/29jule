
# Первый файл по контролю версий
- Заголовки задаются через параметр # 
- Заголовки нижних уровней (подзаголовки) задаются через параметр ## **(пример ниже)**

## Выделение текста:
*Курсив.* - для выделения текста курсивом, его закрывают между * * (без пробелов между текстом и звездочками)

**Полужирный текст.** - для выделения текста полужирным шрифтом, его закрывают между ** ** (без пробелов между текстом и звездочками)

## Создание списков:
### Пример создания нумерованных списков:
* 1. Элемент 1
* 2. Элемент 2
* 3. Элемент 3
- *Чтобы создать нумерованный список необходимо поставить * пробел и цифру начала отсчета, точка, пробел и сам необходимый нам текст*

- Элемент 1
- Элемент 2
- Элемент 3
- *Чтобы создать ненумерованный список необходимо поставить тире, пробел и сам необходимый нам текст*

## Приклепление ссылки
*Полезную информацию для самостоятельного изучения и саморазвития рекомендуется представлять в предметном указателе в виде "Полезных ссылок"*

**Например:** https://gbcdn.mrgcdn.ru/uploads/asset/4245110/attachment/d4eb8c232f8f2bdf4e42ba7cb49e0c50.pdf
- Для вставки "Полезной ссылки" достаточно её скопировать из адресной строки браузера и вставить в пустое поле выбранного раздела или части текста.

Однако это неудобно потому рекоммендуется заместить ссылку маской [С. Чакон, Б. Штауб. Git для профессионального программиста](https://gbcdn.mrgcdn.ru/uploads/asset/4245110/attachment/d4eb8c232f8f2bdf4e42ba7cb49e0c50.pdf)

## Цитирование

- Для создания цитат используют символ > "тескт"

*Например:*
> Учиться, учиться и еще раз учиться (Ленин В.И.)

## Работа с изображениями (картинками)
Чтобы вставить изображения надо использовать следующий синтаксис: "! затем, для описания картинки [], и вконце () где между ними написать имя картинки расположенной в данной папке".
